def createRandomMultiNetwork(type, N, L, p1=0.4, p2= 0.1, w=None, verbose=True):
  M = np.zeros((N,N,L,L))
  if 'w' in type: assert w, "Error: 'w' cannot be empty!"
  for k in range(L):
    for l in range(L):
      for i in range(N):
        for j in range(N):
          # 判断是否为同层
          if k == l and np.random.rand() < p1:
            # 这里我们不允许同层同顶点之间的连接(即不存在自环)
            if i != j and type == 'uu':
              M[i][j][k][l] = M[j][i][k][l] = 1
            elif i != j and type == 'du':
              M[i][j][k][l] = 1
            elif i != j and type == 'uw':
              M[i][j][k][l] = M[j][i][k][l] = np.random.randint(w)
            elif i != j and type == 'dw':
              M[i][j][k][l] = np.random.randint(w)
          elif k != l and np.random.rand() < p2:
            # 这里我们允许不同层相同顶点之间的连接
            if type == 'uu':
              M[i][j][k][l] = M[j][i][k][l] = 1
            elif type == 'du':
              M[i][j][k][l] = 1
            elif type == 'uw':
              M[i][j][k][l] = M[j][i][k][l] = np.random.randint(w)
            elif type == 'dw':
              M[i][j][k][l] = np.random.randint(w)
          else:
            # 不存在边的顶点对
            if not verbose:
              print('i,j,k,l = ', i, j, k, l)
  return M

# 获得具体某一层的2阶正交基张量
def ct(d, rank):
  """
  ct = canonical tensor
  """
  E = np.zeros((rank,rank))
  E[d][d] = 1

# a function that selects the cycles
def sc(h,k,l,num_layer):

def ten2mat(t, mode):
    """
    a tensor unfolding

    Parameters:
    ---
    `t`: a tensor object

    `mode`: <int> 1-mode = 0, 2-mode = 1,...
    """
    return np.reshape(np.moveaxis(t, mode, 0), (t.shape[mode], -1), order = 'F')

def mat2ten(mat, tensor_size, mode):
    """
    Examples
    ---
    >X = np.array(
        [[[1, 2, 3, 4], [3, 4, 5, 6]],
        [[5, 6, 7, 8], [7, 8, 9, 10]],
        [[9, 10, 11, 12], [11, 12, 13, 14]]])
    >print('mode-1 tensor unfolding:')
    >print(ten2mat(X, 0))
    >print('mode-1 tensor folding:')
    >print(mat2ten(ten2mat(X, 0), np.array([3, 2, 4]), 0))
    """
    index = list()
    index.append(mode)
    for i in range(tensor_size.shape[0]):
        if i != mode:
            index.append(i)
    return np.moveaxis(np.reshape(mat, list(tensor_size[index]), order = 'F'), 0, mode)

2. 度中心性（Degree centrality）

We obtain a multidegree centrality vector

\[ K^{\alpha}=M_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}U_{\tilde{\gamma}}^{\tilde{\delta}}u^{\beta}\tag{2.1} \]

the above equation is equal to:

\[ K^{\alpha}=\sum_{\tilde{h},\tilde{k}=1}^{L}k^{\alpha}(\tilde{h}\tilde{k})\tag{2.2} \]

where \( k^{\alpha}(\tilde{h}\tilde{k}) \) is the degree centrality vector that corresponds to connections between layers \( \tilde{h}, \tilde{k} \). The definitions of mean degree, second moment, and variance are analogous to the corresponding monoplex-network counterparts, except that one uses \( K^{\alpha} \)instead of \( k^{\alpha} \).

3.聚集系数（Clustering coefficients）

For multilayer networks, the triangles can be closed using intralayer connections from different layers. We define a global clustering coefficient on a multilayer adjacency tensor by generalizing Eq. (3.3) in here for fourth-order tensors:

\[ c(M_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}})=\mathcal{N}^{-1}\frac{M_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}M_{\epsilon\tilde{\eta}}^{\beta\tilde{\delta}}M_{\alpha\tilde{\gamma}}^{\epsilon\tilde{\eta}}}{M_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}F_{\epsilon\tilde{\eta}}^{\beta\tilde{\delta}}M_{\alpha\tilde{\gamma}}^{\epsilon\tilde{\eta}}}\tag{3.1} \]

where we again define \( F_{\epsilon\tilde{\eta}}^{\beta\tilde{\delta}}=U_{\epsilon\tilde{\eta}}^{\beta\tilde{\delta}}-\delta_{\epsilon\tilde{\eta}}^{\beta\tilde{\delta}} \) as the adjacency tensor of a complete multilayer network (without self-edges). The choice of the normalization factor \( \mathcal{N} \) is again arbitrary, but the choice \( \mathcal{N}=\max_{\alpha,\beta,\tilde{\gamma},\tilde{\delta}}\left\{M_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}\right\} \) ensures that Eq. (3.1) is well defined for both weighted and unweighted multilayer networks.

U = np.einsum('i,j,k,l->ijkl', np.ones(N), np.ones(N), np.ones(L), np.ones(L))
# 同层同顶点等于1，其他都等于0,即层L=L，顶点N=N
kron_delta = np.einsum('ij,kl->ijkl', np.eye(N), np.eye(L))
F = U - kron_delta
M_max = np.max(M)
c = np.einsum('ijkl,jmln,mink', M, M, M) / np.einsum('ijkl,jmln,mink', M, F, M) / M_max

A more traditional, and also simpler, approach to calculating a global clustering coefficient of a multilayer network is to project it onto a single weighted network.

\[ c(O_{\beta}^{\alpha})=\mathcal{M}^{-1}\frac{M_{\beta\tilde{\gamma}}^{\alpha\tilde{\gamma}}M_{\epsilon\tilde{\delta}}^{\beta\tilde{\delta}}M_{\alpha\tilde{\eta}}^{\epsilon\tilde{\eta}}}{M_{\beta\tilde{\gamma}}^{\alpha\tilde{\gamma}}F_{\epsilon\tilde{\delta}}^{\beta\tilde{\delta}}M_{\alpha\tilde{\eta}}^{\epsilon\tilde{\eta}}}\tag{3.2} \]

where \( \mathcal{M}=\max_{\alpha,\beta}\left\{M_{\beta\tilde{\gamma}}^{\alpha\tilde{\gamma}}\right\}/L \). In general, the clustering coefficient in Eq. (3.1) is different from the one in Eq. (3.2) because Eq. (3.2) discards all interlayer connections. These equations yield the same value when there is only a single layer or, more generally, when there are no interlayer edges and all of the intralayer networks are exactly the same.

# 计算聚集系数

The global clustering coefficient defined in Eq. (3.2) sums the contributions of three-cycles for which each step of a walk is on the same layer and those for which a walk traverses two or three layers. We decompose this clustering coefficient to separately count the contributions of three-cycles that take place on one, two, and three layers [47]. Using our tensorial framework, we modify Eq. (3.2) as follows:

\[ c(M_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}},w^{\Omega})=\frac{M_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}M_{\gamma\tilde{\sigma}}^{\beta\tilde{\rho}}M_{\alpha\tilde{\epsilon}}^{\gamma\tilde{\eta}}\sum_{\tilde{h},\tilde{k},\tilde{l}=1}^LE_{\tilde{\gamma}}^{\tilde{\delta}}(\tilde{h}\tilde{h})E_{\tilde{\rho}}^{\tilde{\sigma}}(\tilde{k}\tilde{k})E_{\tilde{\eta}}^{\tilde{\epsilon}}(\tilde{l}\tilde{l})\delta_{\Omega}(\tilde{h},\tilde{k},\tilde{l})w^{\Omega}}{M_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}F_{\gamma\tilde{\sigma}}^{\beta\tilde{\rho}}M_{\alpha\tilde{\epsilon}}^{\gamma\tilde{\eta}}\sum_{\tilde{h},\tilde{k},\tilde{l}=1}^LE_{\tilde{\gamma}}^{\tilde{\delta}}(\tilde{h}\tilde{h})E_{\tilde{\rho}}^{\tilde{\sigma}}(\tilde{k}\tilde{k})E_{\tilde{\eta}}^{\tilde{\epsilon}}(\tilde{l}\tilde{l})\delta_{\Omega}(\tilde{h},\tilde{k},\tilde{l})w^{\Omega}}\tag{3.3} \]

and \( w^{\Omega} \) is a vector that weights the contribution of three-cycles that span \( \Omega \) layers, the choice \( w^{\Omega}=(1/3,1/3,1/3) \), so that it takes into account that there might be some cost for crossing different layers. \( \delta_{\Omega}=(\delta_1(\tilde{h},\tilde{k},\tilde{l}),\delta_2(\tilde{h},\tilde{k},\tilde{l}),\delta_3(\tilde{h},\tilde{k},\tilde{l})) \) (where \( \Omega=1,2,3 \)) is a function that selects the cycles with \( \Omega \) layers:

\[ \begin{aligned} \delta_1(\tilde{h},\tilde{k},\tilde{l})&=\delta_{\tilde{h}\tilde{l}}\delta_{\tilde{h}\tilde{k}}\delta_{\tilde{k}\tilde{l}}\\ \delta_2(\tilde{h},\tilde{k},\tilde{l})&=(1-\delta_{\tilde{h}\tilde{l}})\delta_{\tilde{h}\tilde{k}}+(1-\delta_{\tilde{k}\tilde{l}})\delta_{\tilde{h}\tilde{k}}+(1-\delta_{\tilde{h}\tilde{k}})\delta_{\tilde{k}\tilde{l}}\\ \delta_3(\tilde{h},\tilde{k},\tilde{l})&=(1-\delta_{\tilde{h}\tilde{l}})(1-\delta_{\tilde{k}\tilde{l}})(1-\delta_{\tilde{h}\tilde{k}}) \end{aligned} \]

# if w = (1/3, 1/3, 1/3)
w = np.array([1/3, 1/3, 1/3])
c1 = np.zeros((L,L,L,L,L,L))

# 计算聚集系数

cd = np.einsum('ijklmn,jilknm', np.einsum('ijkl,jomn,oipq', M, F, M), c1)
c = cn / cd

4.特征向量中心性（Eigenvector centralities）

We can define a square supra-adjacency tensor in \( \mathbb{R}^{NL \times NL} \), Such supramatrices correspond to unique unfoldings of corresponding fourth-order tensors. According to this unique mapping, if \( \lambda_1 \) is the largest eigenvalue and \( V_{\alpha\tilde{\gamma}} \) is the corresponding eigentensor, then it follows that

\[ M_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}V_{\alpha\tilde{\gamma}} = \lambda_1V_{\beta\tilde{\delta}} \]

The monoplex notion of eigencentrality grants impor- tance to a node based on its connection to other nodes. One needs to be careful about both intralayer and interlayer connections when interpreting the results of calculating a multilayer generalization of it. Moreover, by considering an overlay monoplex network or a projection monoplex network, it is possible to derive separate centrality scores for different layers.

5.接近中心性、介数中心性和随机游走（Betweenness and Closeness centrality）

一般地，接近中心性和介数中心性的计算都是基于最短路径的。但在现实中，信息不一定沿最短路径传播，而可能遵循随机游走的路径传播。参考这篇文献，我们将实现随机游走中心性。

Random walk occupation centrality

随机游走占有率，该指标衡量在整个网络随机游走中达到稳定态时，游走达到各个顶点的概率。概率越大，该顶点越重要。

\[ T_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}\Pi_{\alpha\tilde{\gamma}}=\lambda\Pi_{\beta\tilde{\delta}} \]

where \(T_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}\) is the tensor of transition probabilities for jumping between pairs of nodes and switching between pairs of layers. \(u^{\alpha\tilde{\gamma}}=T_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}u^{\beta\tilde{\delta}}\)

the transition tensor is given by:

\[ T_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}=M_{\rho\tilde{\sigma}}^{\alpha\tilde{\gamma}}\tilde{D}_{\beta\tilde{\delta}}^{\rho\tilde{\sigma}} \]

\[ S^{\alpha\tilde{\gamma}}=M_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}U^{\beta\tilde{\delta}} \]

Note that \(S_{\beta\tilde{\delta}}=M_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}U_{\alpha\tilde{\gamma}}\) if M is a directed and weighted adjacency tensor, and in an undirected network \(S_{\beta\tilde{\delta}}=S^{\alpha\tilde{\gamma}}\)

where \( \tilde{D}_{\beta\tilde{\delta}}^{\rho\tilde{\sigma}} \) is the tensor whose entries are the the non-zero inverse of the strength tensor \( D_{\beta\tilde{\delta}}^{\rho\tilde{\sigma}} \) whose entries are all zeros, except for \(\rho=\beta\) and \(\tilde{\sigma}=\tilde{\delta}\) where the entries are given by \(s^{\alpha\tilde{\gamma}}\)(the strength of node \(\alpha\) in layer \(\tilde{\gamma}\)). \(M_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}\) is the weighted adjacency tensor.

Betweenness centrality

Definition: the random walk betweenness of a node j is defined as the amount of random walks between any pair (o, d) of nodes that pass through j. The absorbing transition tensor on a particular node d by:

\[ (T_{[d]})_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}=\left\{ \begin{aligned} 0&, \beta=d\\ T_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}&, \text{else} \end{aligned} \right. \]

the average number of times a random walk is given by:

\[ (\tau_{[d]})_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}=\left[(\delta - T_{[d]})^{-1} \right]_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}} \]

and we average over all possible starting layers:

\[ (\tau_{[d]})_{\alpha}^{\beta}=\frac{1}{L}(\tau_{[d]})_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}u^{\tilde{\delta}}u_{\tilde{\gamma}} \]

The overall betweenness centrality is obtained by averaging over all possible origins and destinations:

\[ \tau_{\alpha}=\frac{1}{N(N-1)}\sum_{d=1}^N(\tau_{[d]})_{\alpha}^{\beta}u_{\beta} \]

# absorbing transition tensor
def att(d, T):
  T_a = T.copy()
  L1, L2 = T_a.shape[2:]
  for k in range(L1):
    for l in range(L2): T_a[d,:, k, l] = 0
  return T_a

# the inverse of four-order tensor
def inv(T):
  """
  必须是可以转化为方阵的4阶张量，比如(10,10,5,5)--->(50,50);
  """
  K, L = T.shape[1:3]
  squ_matrix = np.einsum('ijkl->ikjl', T).reshape((K*L, K*L),order='F')
  t = np.linalg.inv(squ_matrix)
  return np.einsum('ijkl->ikjl', t.reshape((K,L,K,L), order='F'))
# 顶点数N

Closeness centrality

Definition: the random walk closeness centrality of a node is defined as the inverse of the average number of steps that a random walker, starting from any other node in the multilayer network, requires to reach i for the first time.

the probability of visiting node \(\beta\) in layer \(\tilde{\delta}\), after t time steps, considering that the walk originated in node \(\alpha\) in layer \(\tilde{\gamma}\):

\[ P_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}(t)=(T_{[d]}^t)_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}} \]

The probability that the walker is absorbed in some node d at a time h equal or smaller than t:

\[ (q_{[d]})^{\alpha\tilde{\gamma}}(h \le t)=u^{\alpha\tilde{\gamma}}-(T_{[d]})_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}u^{\beta\tilde{\delta}} \]

the probability that the first passage time for node d is exactly t by:

\[ (q_{[d]})^{\alpha\tilde{\gamma}}(h=t)=(q_{[d]})^{\alpha\tilde{\gamma}}(h \le t)-(q_{[d]})^{\alpha\tilde{\gamma}}(h \le t-1)=\left [ (T_{[d]}^t)-(T_{[d]}^{t-1})\right ]_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}u^{\beta\tilde{\delta}} \]

the mean first passage time to node d:

\[ (H_{[d]})^{\alpha\tilde{\gamma}}=\sum_{t=0}^{\infty}(q_{[d]})^{\alpha\tilde{\gamma}}(h=t)=\left [ (\delta-T_{[d]})^{-1} \right ]_{\beta\tilde{\delta}}^{\alpha\tilde{\gamma}}u^{\beta\tilde{\delta}} \]

The average mean first passage time \(h_{[d]}\) to node d is obtained by averaging \((H_{[d]})^{\alpha\tilde{\gamma}}\) over all possible starting nodes and layers as:

\[ h_{[d]}=\frac{1}{(N-1)L}u_{\alpha\tilde{\gamma}}(H_{[d]})^{\alpha\tilde{\gamma}}+\frac{1}{N}\pi_{[d]}^{-1} \]

Note that \(\pi_i=\Pi_{i\alpha}u^{\alpha}\).

PageRank算法

最简单的PageRank算法：

\[ V_1 = MV_0 \]

def pagerank(M, N, T=300, tol=1e-6):
    R0 = np.ones(N) / N
    for _ in range(T):
        R = np.dot(M, R0)
        if np.linalg.norm(R - R0) < tol: break
        R0 = R.copy()
    return R

M为转移矩阵，列向相加等于1，\(V_0\)为上网者在每一个网页的概率都是相等的,即1/n。

上述上网者的行为是一个马尔科夫过程的实例，要满足收敛性，需要具备一个条件，即图是强连通的，即从任意网页可以到达其他任意网页。另外，有些网页不存在指向其他网页的链接，但存在指向自己的链接。为此，假设查看当前网页的概率为a，那么他从浏览器地址栏跳转的概率为(1-a)，则

马尔可夫过程按照其状态和时间参数是否连续或者离散分为三种：1.时间和状态都离散的叫做马尔科夫链，2.时间和状态都是连续的叫做马尔科夫过程，3.时间连续，状态离散的叫做连续时间的马尔科夫链。

\[ V_1 = \alpha MV_0+\frac{(1-\alpha)}{N}u \]

def pagerank_google(M, N, alpha=0.8, T=300, tol=1e-6):
    R0 = np.ones(N) / N
    for _ in range(T):
        R = alpha * np.dot(M, R0) + (1-alpha) * np.ones(N) / N
        if np.linalg.norm(R - R0) < tol: break
        R0 = R.copy()
    return R

the Page Rank centrality can be seen as the steady-state solution of the random walk master equation governed by the transition tensor,where the walker jumps to a neighbor with rate r and teleport to any other node in the network with rate 1 − r. This rank-4 tensor is given by:

\[ R_{i\beta}^{j\alpha}=rT_{i\beta}^{j\alpha}+\frac{(1-r)}{NL}u_{i\beta}^{j\alpha} \]

Note that \(NL = N \times L\)

# the transition tensor. 一般r = 0.8, N为顶点数，L为层数.
R = r * T + (1-r) * np.ones_like(T) / (N * L)
